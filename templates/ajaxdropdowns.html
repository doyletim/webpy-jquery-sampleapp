On initial load, this page has static content defined in the 'countries' dropdown.

Selecting a country will perform three ajax calls:

  • The 'Regions from JSON' picker is populated using JSON and Javascript
  • The 'Regions from HTML' picker is populated using HTML from the server
  • A generic message is returned from the server.

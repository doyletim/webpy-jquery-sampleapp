Dropdowns

Dynamically populated dropdowns with web.py on the server-side and jQuery on the client-side. The first dropdown is plain static HTML in this page, the second fetches a JSON object from the server, and the third dropdown fetches plain HTML from the server..

Like this...

On initial load, this page has static content defined in the Countries picker.
Selecting a country will perform three ajax calls:

  • Regions picker is populated using return json.dumps()
  • Regions 2 picker is populated using return "<html>...</html>"
  • A generic message is returned from the server for example, datetime()

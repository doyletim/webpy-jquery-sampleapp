This sample app demonstrates how to deliver an HTML/jQuery web application, using by Python 3.7, web.py and jQuery.
I plan to add more ways to use ajax in the future, but for now we've got dropdowns.
Version information:

  • Python 3.7.3
  • web.py 0.6.2
  • jQuery 3.2.1
  • Popper.js 1.12.9
  • Bootstrap 4

Most pages in this app (including this one) are delivered using web.py templates and the render() method.
However, keeping with the spirit of jQuery and Ajax, web.py templates aren't used beyond delivery of the page. Content is dynamic... that's the point.
